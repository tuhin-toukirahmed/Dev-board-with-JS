Question-1: What are the different ways to select an element in the DOM?

Answer: There are many ways to select an element in the DOM we can use getElementById method, getElementsByClassName method, getElementsByTagName method,querySelector method,querySelectorAll method.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: The difference between innerHTML, innerText, and textContent is that innerHTML returns the HTML code of the element, innerText returns the text content of the element, and textContent returns the text content of the element without any HTML tags.

Question-3: What is event delegation in the DOM?

Answer: Event delegation in the DOM is a technique where we attach a single event listener to a parent element, and then handle events for child elements using event delegation.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling in the DOM is a concept where events are propagated from child elements to parent elements, and then to the document element.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: To create, add, and remove elements using JavaScript, we can use the document.createElement method, appendChild method, and removeChild method.
